# Hospitality Revenue & Occupancy Analysis

## Overview
This project analyzes hotel booking data to uncover insights related to
occupancy, revenue trends, room categories, and city-wise performance using Python.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook

## Key Insights
- Identified room categories with highest average occupancy
- Analyzed weekday vs weekend occupancy patterns
- Evaluated city-wise revenue contribution
